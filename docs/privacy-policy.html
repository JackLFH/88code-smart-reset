隐私政策

应用名称: 88code 自动重置助手

版本: v1.2.0

生效日期: 2025年1月1日

最后更新: 2025年1月1日

📌 总览

✓ 本扩展不收集任何用户个人数据
✓ 所有数据本地存储在您的浏览器中
✓ API密钥使用军工级加密算法保护
✓ 开源代码，随时可审计

1. 数据收集

本扩展不收集、不存储、不传输任何用户个人数据。

我们不会收集以下任何信息：

  • 个人身份信息（姓名、邮箱、电话等）
  • 浏览历史或网页访问记录
  • 地理位置信息
  • 设备信息或指纹
  • 任何其他个人隐私数据

2. 数据存储

所有数据均通过 Chrome Storage API 存储在您的浏览器本地，包括：

  • 88code API密钥：使用 AES-GCM 算法加密后存储
  • 账号配置：账号名称、启用状态、创建时间等
  • 定时设置：重置时间、时区、通知偏好等
  • 操作日志：本地记录操作历史，方便追溯

重要：这些数据仅存储在您的设备上，不会上传到任何服务器。

3. 数据加密

您的 API 密钥使用 AES-GCM 加密算法进行加密存储，具体包括：

  • 256位密钥长度
  • 随机生成的 IV（初始化向量）
  • 加密认证标签（确保数据完整性）
  • 绝不以明文形式存储

4. 网络请求

本扩展仅向 88code.org 官方API发送请求，用于：

  • 测试API密钥有效性
  • 获取账号使用情况
  • 执行订阅配额重置

我们不会将您的数据发送到任何第三方服务器。

5. 权限说明

本扩展请求的权限及用途：

  • storage: 在浏览器本地存储账号配置和设置
  • alarms: 实现定时自动重置功能
  • notifications: 显示重置成功/失败的桌面通知
  • host_permissions (88code.org): 调用88code官方API接口

我们不会请求以下敏感权限：

  • ❌ 访问所有网站数据
  • ❌ 读取浏览历史
  • ❌ 修改网页内容
  • ❌ 管理下载内容

6. 数据安全承诺

  • ✅ 所有数据仅存储在用户本地浏览器
  • ✅ API密钥使用工业级加密算法（AES-GCM）
  • ✅ 不与任何第三方共享数据
  • ✅ 开源代码，任何人都可以审计
  • ✅ 符合GDPR等国际隐私法规

7. 用户控制

您拥有对数据的完全控制权，可以随时：

  • 在扩展设置中删除已保存的账号
  • 清空所有操作日志记录
  • 卸载扩展（所有本地数据将被自动删除）

8. 儿童隐私

本扩展不针对13岁以下儿童。我们不会故意收集儿童的个人信息。

9. 政策更新

本隐私政策可能会不定期更新。重大变更将通过以下方式通知用户：

  • 在扩展更新日志中说明
  • 在GitHub仓库中发布公告
  • 更新本页面的"最后更新"日期

10. 开源透明

本扩展采用 MIT 开源协议，所有源代码托管在 GitHub：
https://github.com/JackLFH/88code-smart-reset

我们欢迎任何人审计代码，提出安全建议，或贡献改进。
